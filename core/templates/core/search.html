Detailed Search

Medieval Architects is an interactive map and database of medieval architectural records, designed to make the historic architecture of the period accessible and engaging for researchers and the public.

This project brings those records to life with modern mapping and search tools.

  • Explore records by location and type
  • Add and edit entries (registered users)
